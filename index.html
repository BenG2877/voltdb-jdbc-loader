Voltdb-jdbc-loader

A command-line tool to load data into VoltDB from a JDBC source.

Download this project as a .zip file Download this project as a tar.gz file

What does it do?

This project can be used to connect to any JDBC database and transfer entire tables into a VoltDB database that has an equivalent table defined. It works one table at a time, and it assumes that you have already built the equivalent schema in VoltDB.
